# Nova Assistant Pro

A modular, feature-rich voice assistant for Windows.

## Features
- **App Launcher**: Open common Windows apps, browsers, and dev tools.
- **Web Search**: Search Google and YouTube directly via voice.
- **Knowledge**: Wikipedia integration for quick information.
- **System Controls**: Adjust system volume (Up/Down/Mute).
- **Wake Word**: Privacy-focused wake word detection using Porcupine.

## Installation
1. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
2. Ensure `hey_nova.ppn` is present in the root directory.
3. Run the assistant:
   ```bash
   python main.py
   ```

## Configuration
Update `config.yaml` to add custom application paths or adjust settings.
